Detection of Fake Bank Currency with Machine Learning Algorithms: A Hybrid Multi-Modal Approach

Debobrato Das1*, Imran Mahmud2
1Department of Software Engineering, Daffodil International University, Dhaka, Bangladesh
2Forensic Research Lab, Dhaka, Bangladesh
Abstract The proliferation of sophisticated counterfeit currency poses a severe threat to global economic stability. Traditional detection methods often fail to identify high-quality replicas. This research presents a hybrid multi-modal framework that integrates Self-Supervised Anomaly Detection with Squeeze-and-Excitation (SE) Attention Mechanisms. By utilizing a Deep Autoencoder, the system identifies forgeries by calculating the Forensic Reconstruction Error ($L_{rec}$). Experimental results on Bangladeshi currency demonstrate a superior accuracy of 98.8% for the combined hybrid model.
KEYWORDS Deep Learning; Computer Vision; Anomaly Detection; Forensic Analysis.

1. Introduction

Counterfeit currency has remained a critical challenge since the inception of monetary systems. Modern forgers utilize high-precision scanning tools to bypass standard UV and magnetic sensors. This paper proposes a system that doesn't just look for "fake" patterns but learns the "perfect distribution" of a genuine note using Deep Learning.

1.1 Motivation

The motivation arises from the need for a scalable, low-cost solution for identifying "Supernotes" in real-world environments. By leveraging Machine Learning, we can transform standard mobile sensors into forensic-grade tools.

2. Methodology

The Elite Forensic Intelligence framework operates in two domains:

2.1 Forensic Domain (Deep Learning)

We utilize the CurrencyForensicNet architecture. Anomaly detection is based on the Reconstruction Discrepancy:

\[L_{rec} = || X_{original} - X_{reconstructed} ||^{2}\]

The addition of SE-Blocks allows the model to dynamically focus on spatial regions containing security watermarks and micro-printing.

2.2 Structural Domain (Computer Vision)

Structural integrity is verified using Hough Line Transforms for security threads and Laplacian variance for intaglio print quality verification.
